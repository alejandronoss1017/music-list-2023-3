# Configuration

1. Install [Node.js](https://nodejs.org/en/download)
2. Run the command:

```bash
  npm install
```

3. Run the command:

```bash
  npm install -g @angular/cli
```


# Usage

Go to the front-end directory

```bash
  cd music-list-2023-3/front-end
```

Start the server

```bash
  ng serve -o
```
